## Description

Catalyst example solver is a solver built for the Catalyst intent system. It uses [Nest](https://github.com/nestjs/nest) as a framework.

The Catalyst example solver is not a reference solver and is not production ready. It is meant as an example of the general flow for getting and filling intents in the Catalyst intent system as a solver.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
